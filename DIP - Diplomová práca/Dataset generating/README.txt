Script for inserting warts into fingerprints
Author: xhalin01

Requirements: python 3.6.8 , other described in requirements.txt 

Usage:
- place fingerprints into 'fingerprints' folder
- run script
- damaged fingerprints are in the 'damaged' folder, non-damaged processed fingerprints are located in 'pure' folder

Note:
- warts were generated with SyFDaS: https://www.fit.vut.cz/research/product/600/.cs
